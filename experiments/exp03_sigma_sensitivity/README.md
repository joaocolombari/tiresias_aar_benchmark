# Experiment 3: Sigma Sensitivity

Experiment 3 evaluates how the Gaussian attention width `sigma_deg` changes
static source enhancement.

It uses the same measured BRIRs, source geometry and 100 deterministic
LibriSpeech pairs used in Experiments 4 and 5.

## Question

How does sigma trade angular selectivity against robustness to head-orientation
error?

## Geometry

- source A: `-30 deg`;
- source B: `+30 deg`;
- measured binaural responses from Experiment 2;
- static head yaw grid from `-90 deg` to `+90 deg`;
- monophonic attention model with one scalar gain per source.

The earlier `45 deg` geometry is not used.

## Method

For every head yaw and sigma, the benchmark evaluates two independent target
definitions:

- target A: source A is desired;
- target B: source B is desired.

The target is not chosen from the current head hemisphere. This avoids a
circular evaluation where the same yaw both selects the target and computes the
attention gain.

The output is:

```text
y_L = g_A * x_A,L + g_B * x_B,L
y_R = g_A * x_A,R + g_B * x_B,R
```

where `g_A` and `g_B` are monophonic source gains, applied equally to both ears
of each measured binaural source image.

## Required Inputs

- BRIRs from Experiment 2:
  `experiments/exp02_brir_measurement/processed/exp02_campaign_20260718_001/irs/`
- LibriSpeech subset:
  `datasets/librispeech_dev_clean_200_seed_2026/`

The default config uses 100 deterministic, non-overlapping source pairs from
the 200-file subset:

```yaml
pair_count: 100
pair_seed: 20260720
```

## Run

From the repository root:

```bash
PYTHONPATH=src .venv/bin/python -m tiresias_benchmark experiment-run \
  --experiment 3 \
  --config experiments/exp03_sigma_sensitivity/config.yaml
```

On machines without the project virtual environment, use:

```bash
PYTHONPATH=src python3 -m tiresias_benchmark experiment-run \
  --experiment 3 \
  --config experiments/exp03_sigma_sensitivity/config.yaml
```

## Default Grid

```yaml
head_yaw_deg: [-90, -80, ..., 80, 90]
sigma_deg: [10, 20, 30, 45, 60]
```

With 100 speech pairs and two target definitions, this produces:

```text
19 head-yaw positions
x 5 sigma values
x 2 target definitions
x 100 speech pairs
= 19,000 detailed rows
```

## Figures

`exp03_sigma_tir_heatmaps` shows TIR improvement over the static operating
surface. Each panel is one target definition, the x-axis is head yaw and the
y-axis is sigma.

`exp03_sigma_gain_and_tir_curves` shows the target-A case as signed head yaw
relative to the target direction. The dashed vertical line marks the target
direction and the dotted vertical line marks the interferer direction. The left
panel shows the target/interferer gain contrast generated by the attention
model. The right panel shows the corresponding measured-BRIR audio impact.

## Interpretation Notes

- Smaller sigma creates a narrower attention field and higher source contrast
  near the attended source.
- Larger sigma creates a broader attention field and lower peak selectivity.
- The attention model is monophonic.
- Measured BRIRs provide the binaural source images.
- Analytical ITD/ILD rendering is not used.
